# Projet-PFE--CVA-Pricing

## Overview

This repository contains the Python implementation developed during my internship project at Société Générale.

The objective of this project is to investigate different approaches for reducing the computational cost of Credit Valuation Adjustment (CVA) calculations while maintaining a high level of accuracy.

## Implemented Methods

- Monte Carlo simulation under the Vasicek interest rate model
- Polynomial approximation
- Quasi-Monte Carlo using Sobol sequences
- Neural Network surrogate model

## Repository Structure

```
.
├── MC.py          # Monte Carlo CVA implementation
├── Poly.py        # Polynomial approximation
├── QMC.py         # Sobol Quasi-Monte Carlo implementation
├── NN.py          # Neural Network surrogate model
├── Bib.py         # Libraries

```
